# ML 튜터 작업지시서 (v0.1 완료 버전)
## 데이터 기반 기계학습 모델 구현을 위한 AI 튜터 & 멘토형 서비스
> v0.1 MVP 완료 기준 | 2026-05-20

---

## 1. 프로젝트 개요

본 서비스는 고등학교 정보 교과에서 데이터 기반 기계학습 모델 구현을 학습하는 학생들을 위한
AI 튜터 및 멘토형 웹 서비스입니다.

### 핵심 철학
- 정답 코드를 직접 제공하지 않음 — 인지적 비계(Scaffolding) 방식으로 학생이 스스로 해결하도록 유도
- 기존 AI 챗봇과의 차별점: 현재 수업 단계와 미션 맥락을 인식한 구조화된 AI 튜터
- 과정 중심 평가: 결과물뿐 아니라 학습 여정 전체를 기록 및 평가

### 3-4. Pyodide 파이썬 실행 ★ 추가됨

```
- 라이브러리: Pyodide v0.25.1 (CDN)
- 지원 패키지: pandas, numpy
- 미지원: matplotlib (브라우저 환경 제약) → print() 출력으로 대체
- CSV 파일: 학생이 직접 업로드 → Pyodide 가상 파일 시스템에 마운트
- 파일 유지: 현재 세션 내에서만 유지 (새로고침 시 초기화)
          → v0.2 Firebase Storage 연동 후 차시 간 파일 유지 가능
- 초기 로딩: 최초 실행 시 20~30초 소요 (패키지 다운로드)
- stdout 캡처: io.StringIO()로 print() 출력 캡처 후 결과창에 표시
- 코드 템플릿: 모든 출력에 print() 명시 (df.head() → print(df.head()))
```

### 3-5. 3단계 샘플 콘텐츠 (데이터 전처리)

#### 학습 (3개 개념)
1. 결측치(NaN) — `isnull()`, `dropna()`, `fillna()`
2. 이상치(Outlier) — `describe()`, IQR 방식
3. 데이터 타입 변환 — `dtypes`, `astype()`, `LabelEncoder()`

#### 퀴즈 (3문제)
1. 결측치 확인 함수 → `isnull()` (정답: 0번)
2. 결측치 행 제거 함수 → `dropna()` (정답: 1번)
3. 결측치 채우기 함수 → `fillna()` (정답: 2번)

#### 실습 미션 (5개, 순차 해제)
| 미션 | 제목 | 핵심 함수 |
|------|------|---------|
| 1 | CSV 파일 불러오기 | `pd.read_csv()` |
| 2 | 데이터 구조 확인 | `df.shape`, `df.info()`, `df.head()` |
| 3 | 결측치 확인 및 처리 | `isnull().sum()`, `dropna()` or `fillna()` |
| 4 | 이상치 탐색 | `df.describe()` |
| 5 | 데이터 타입 확인 및 변환 | `df.dtypes`, `astype()` |

## 5. 개발 주의사항

### Claude API
- 모델: `claude-sonnet-4-20250514` 고정
- `max_tokens`: 1000 고정
- API 키: `.env` → `REACT_APP_CLAUDE_API_KEY`
- **MVP**: `anthropic-dangerous-direct-browser-access: true` 헤더로 직접 호출
- **v1.0 배포 시**: 백엔드 서버 필요 (API 키 노출 방지)

### Pyodide
- `public/index.html`에 CDN 스크립트 추가 필수
- `matplotlib` 사용 불가 → 코드 템플릿에서 제외
- 모든 출력은 `print()` 명시 필요 (`df.head()` 단독 사용 시 출력 없음)
- CSV 파일은 세션 내에서만 유지 (v0.2에서 Firebase Storage로 영구 저장)

### 저장 관련
- MVP: React `useState`만 사용 (`localStorage` 금지)
- v0.2+: Firebase Firestore + Storage 사용

### UI/UX
- `position: fixed` 사용 금지
- CSS 변수로 색상 관리
- 퀴즈 통과 전 실습 탭 잠금
- 미션 순차 해제 (이전 미션 완료 후 다음 미션 활성화)
